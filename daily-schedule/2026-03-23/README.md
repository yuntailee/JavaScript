3. 先用手寫「不用 map」的版本，再改成 map/filter，對照差異。

## 作業

1. 寫一個函式 `isAdult(age)`，滿 18 回傳 `true`，否則 `false`。
2. 有一個數字陣列 `[12, 19, 5, 30]`，用 `filter` 留下「成年」的年齡；或用 `map` 把每個數字變成字串 `"12 歲"`（擇一）。
3. 在檔案最上方用註解寫：今天選 `map` 還是 `filter`，為什麼。
