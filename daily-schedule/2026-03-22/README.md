## 建議學習方式

1. 開瀏覽器 **開發者工具**（F12）→ **Console**。
2. 閱讀 MDN：[變數](https://developer.mozilla.org/zh-TW/docs/Learn/JavaScript/First_steps/Variables)（約 15～20 分鐘）。
3. 在 Console 裡親自輸入：宣告變數、字串相加、樣板字串。
4. 試錯：故意用 `const` 再指定一次，看錯誤訊息。
