## 今日學習重點

- `JSON.parse`：字串 → 物件
- `JSON.stringify`：物件 → 字串
- 模擬「後端回傳一筆使用者 JSON」

## 建議學習方式

1. 在 Console 試：假 JSON 字串解析後取 `name`。
2. 物件轉字串後再 `parse` 回來，確認資料還在。

## 作業

1. 常數 `raw = '{"id":1,"name":"Ada","role":"user"}'`，parse 後顯示在頁面上三個欄位。
2. 建立一個物件，用 `stringify` 印出，存成「假 API 回應」字串。
3. 故意寫錯 JSON（少逗號），用 `try/catch` 包住 `parse`，顯示「資料格式錯誤」（選做）。
